Enchanting allows you to add powerful bonuses to your weapons, tools, and armor!

üéÆ ENCHANTING COMMANDS:
- /enchant <item> <enchantment> <level> - Enchant an item

üìä ENCHANTING BASICS:

WHAT IS ENCHANTING:
- Add magical effects to items
- Improves item performance
- Requires lapis lazuli
- Costs increase with level
- Multiple enchants can be combined

HOW IT WORKS:
1. Have item in inventory
2. Gather lapis lazuli
3. Use /enchant command
4. Select enchantment and level
5. Pay the cost
6. Item is enchanted!

REQUIREMENTS:
- Lapis Lazuli (1-64 per enchant)
- Coins (varies by level)
- Enchanting skill level
- Compatible item type
